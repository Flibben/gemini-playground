# Design System

## Colors
* **Primary (Brand):** #053d2b (Buttons, Highlights, Links)
  * *Primary-50:* #f3f5f4 (Backgrounds)
  * *Primary-100:* #e6ecea (Selection)
  * *Primary-600:* #053727 (Hover States)
  * *Primary-900:* #0d1f2a (Dark Accents)
* **Secondary (Dark):** #0f172a (Headings, Footer, Nav)
* **Background:** #f8fafc (Slate-50)
* **Text:** #334155 (Slate-700)
* **Card Bg:** #ffffff

## Typography
* **Font:** Inter, System Sans-serif (Roboto, Helvetica, Arial)
* **Headings:** Bold (700), Tracking-tight.
  * *H1:* 48px (3rem)
  * *H2:* 30px (1.875rem)
  * *H3:* 20px (1.25rem)
* **Body:** 16px (1rem), Line-height 1.625 (Relaxed)

## UI Components
* **Buttons:** Rounded-lg, Medium (2.5rem height), Font-medium.
* **Cards:** White bg, Border slate-200, Rounded-xl, Shadow-sm.
* **Inputs:** Rounded-lg, Border slate-300, Focus ring-2.
